Var

1. Variables are containers for storing data (storing data values)

2. The Var keyword is used in all Javascript code from 1995 to 2015.

3. The let and const keywords were added to Javascript in 2015

4. Var is old way to declare a variable.

5. You can change var value or redeclare also you can update the value.


Let

Let is block scoped - Anything within curly braces is a block

Let can't be accessed out of particular block

Let can be updated but you can't redeclare


Const

Variable declared with the const maintain contant value

You can't redclare const variable

Const variable value is constant, can't change the value.
